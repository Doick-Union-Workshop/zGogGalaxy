# zGogGalaxy

**zGogGalaxy** is a Union plugin that brings [GOG Galaxy](https://www.gog.com/galaxy) platform features directly into Gothic games based on ZenGin, via the [GOG Galaxy SDK](https://docs.gog.com/sdk/). It handles user authentication and achievement management, making it easy for mod developers to integrate cloud-based achievements into their mods.

Originally developed as a tool for [Golden Gate 2: Goddess Heart](https://goldengatemod.com/golden-gate-2) mod, the plugin was based on [GodotGOG](https://github.com/binogure-studio/GodotGOG) by [Binogure Studio](https://github.com/binogure-studio), adapted and extended for the Union plugin framework.

## Supported games

- [Gothic I](https://gothic.fandom.com/wiki/Gothic_1)
- [Gothic Sequel](https://gothic.fandom.com/wiki/Gothic_Sequel)
- [Gothic II](https://en.wikipedia.org/wiki/Gothic_II)
- [Gothic II Night of The Raven](https://en.wikipedia.org/wiki/Gothic_II:_Night_of_the_Raven)
